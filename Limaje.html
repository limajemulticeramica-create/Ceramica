🗄️ Exportar / Importar Datos (JSON)

Exporta todos los datos del sistema (productos, categorías, movimientos, cartera) a un archivo .json para respaldar o transferir a otro dispositivo. También puedes importar un archivo previamente exportado.

📊 Importar inventario desde Excel

Importa solo desde Piso, Azulejos, Porcelanato y Fachaletas del Base de datos sistema 2026.xlsx. Bod.1 → Bodega 1, Bod.2 → Bodega 2. Los productos del Excel tienen códigos tipo IMP-… y categoría el nombre de la hoja; no modifican el catálogo demo (001, Corte Azul…) porque los nombres no coinciden. En el inventario use el filtro Categoría para ver Piso, Azulejos, etc.
