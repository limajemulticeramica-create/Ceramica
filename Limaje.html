🛠 Utilidades

Inactividad → cerrar sesión

Tras los minutos indicados sin actividad (clic, tecla, scroll…), el sistema cierra la sesión de Supabase y vuelve a la pantalla de inicio de sesión. Ya no se usa pantalla de PIN: vuelva a entrar con su correo y contraseña.

El campo de «PIN de desbloqueo» anterior ya no aplica; puede ignorarlo.

🧨 Reset del sistema (solo súper admin)

Esta acción reinicia el sistema en este equipo (catálogo, inventario/lotes, movimientos, cartera, compras, PDFs registrados, auditoría, trabajadores, etc.) y además vacía cotizaciones en la nube (si hay conexión y su rol es configurador).

🔢 Reiniciar numeración de documentos (súper admin)

Reinicia a 1 la numeración de recibos/ventas y cotizaciones. Esta acción limpia únicamente los módulos de documentos para empezar de cero.

👤 Usuarios del Sistema

Importante: aquí están quienes inician sesión con correo y contraseña (rol: vendedor, admin, configurador…). No es el módulo 👷 Trabajadores (ese es solo nómina / pagos internos). Para cambiar roles use el botón Rol (solo administrador o súper admin). Contraseña de otra persona: use Supabase → Authentication → Users o pídale que use ¿Olvidó su contraseña? en el login.

Usuarios Registrados
🗄️ Exportar / Importar Datos (JSON)

Exporta todos los datos del sistema (productos, categorías, movimientos, cartera) a un archivo .json para respaldar o transferir a otro dispositivo. También puedes importar un archivo previamente exportado.

📊 Importar inventario desde Excel

Importa solo desde Piso, Azulejos, Porcelanato y Fachaletas del Base de datos sistema 2026.xlsx. Bod.1 → Bodega 1, Bod.2 → Bodega 2. Los productos del Excel tienen códigos tipo IMP-… y categoría el nombre de la hoja; no modifican el catálogo demo (001, Corte Azul…) porque los nombres no coinciden. En el inventario use el filtro Categoría para ver Piso, Azulejos, etc.
